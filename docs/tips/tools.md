## 🔧 工具

### 壁纸样机生成器

https://phonemockup.top/

上传壁纸，生成带壳的样式，自己做的，还没怎么加功能（需要魔法）

### 微信小游戏制作工具

https://gamemaker.weixin.qq.com/#/

可视化的微信小游戏制作工具，像低代码一样搭出小游戏

### 视频转 Live 实况

美图秀秀

之前一直在找视频转 Live 实况的软件，最后发现美图秀秀上就有这个功能

视频剪辑-编辑-右上角保存的地方点击三个点（更多功能）-选择 Live 实况
